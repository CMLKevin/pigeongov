# Govforms Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a local-first `pigeongov` CLI and shared MCP server for the 2025 federal tax year with deterministic tax math, interactive intake, validation, and JSON/PDF outputs.

**Architecture:** Keep all arithmetic and mapping in a pure engine layer, all form definitions in versioned schemas, and expose the same shared workflows through a human CLI and an xmcp-based MCP surface. Build tax logic test-first from known scenarios before layering prompts, PDF I/O, and MCP tools.

**Tech Stack:** TypeScript, pnpm, commander, @inquirer/prompts, chalk, zod, pdf-lib, pdf-parse, vitest, xmcp.
