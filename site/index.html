Local-first government forms copilot

Plan, review, and validate government paperwork without leaving your machine.

PigeonGov is a privacy-first CLI, TUI, and MCP product for US government forms. It helps with taxes, immigration packets, unemployment claims, healthcare enrollment, business licenses, and permit planning while keeping the final review in human hands.

Workflow catalog

Agent-ready

One core engine, two interfaces

Humans use the CLI and Bubble Tea TUI. Agents use the same workflows over MCP with structured outputs, starter data, flagged fields, and review summaries.

  • Structured workflow registry
  • Stable machine JSON output
  • Flag explanations and evidence checklists
  • Shared local packet review model

Browser tools

Planner + packet reviewer at the root site

The root deployment includes a client-side planner and bundle reviewer so people can explore workflows in the browser without turning the product into a hosted filing service.

Important: PigeonGov does not submit to agencies. It prepares structured bundles and review artifacts so a human can finish the job.
