PigeonGov terminal product

The CLI now exposes both guided human commands and machine-friendly JSON flows with deterministic exit codes, flagged fields, and reusable workflow starter data.

TUI

Bubble Tea workspace

The terminal UI is evolving into a multi-workflow workspace with navigation, evidence readiness, review surfaces, and local packet assembly. Use pigeongov tui to launch it.
